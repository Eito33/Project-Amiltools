    # Structure :
        - Mise en place du modéle MVC
            -> Architecture :
                -> src
                    -> core
                        -> components
                            -> Contient l'ensemble des components lié a l'application
                        -> config
                            -> Contient l'ensemble des fichiers de configuration lié a l'application
                        -> modules
                            -> Contient l'ensemble des modules lié a l'application
                        -> Dispatcher > Le dispatcher est le plus haut niveau de récupération des données transmise. 
                           Chaque données récupérer dans les vue lui sont transmise pour etres dispatcher la ou il y en a besoin
                        -> Router > Le router est la partie de l'application qui gere les routes.
                    
                    -> view
                        -> Contient l'ensemble des Vues lié a l'application
